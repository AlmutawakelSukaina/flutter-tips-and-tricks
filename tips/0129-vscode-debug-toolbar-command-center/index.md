# VSCode Debug Toolbar in Command Center

Did you know?

You can now show the Debug Toolbar in the Command Center in VSCode. 🎉

To do this, add the following line to the settings file:

```json
"debug.toolBarLocation": "commandCenter"
```

Happy coding! 💻
